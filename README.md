# Covariance Matrix Estimation

## Overview

This project implements covariance and correlation matrix estimation in Python using financial return data.

Covariance matrices are a fundamental component of quantitative finance and are widely used in:

- Portfolio Risk Measurement
- Mean-Variance Portfolio Optimization (Markowitz)
- Diversification Analysis
- Volatility Estimation
- Asset Allocation

---

## Objectives

The project aims to:

- Estimate covariance matrices
- Estimate correlation matrices
- Visualize dependence among assets
- Support portfolio risk analysis

---

## Technologies

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn

---

## Outputs

- Covariance Matrix
- Correlation Matrix
- Heatmaps
- Portfolio Risk Insights
